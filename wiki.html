四服定位

四服是魔改生存服，提供更多战斗成长路线与高强度试炼玩法。

  • 70 个全新特殊附魔，搭配原版附魔形成多策略流派。
  • 4 个大型试炼副本，挑战更高，回报更高。
  • 完整领地系统，低成本保护家园。
  • 货币和交易行系统，在线可积累货币并快速流通资源。
  • 丰富头衔系统，展示个性与里程碑。
  • 全新公会系统，支持组队冲榜。
  • 等级系统帮助玩家应对试炼怪物。

GMI 纯净生电服

群昵称格式：游戏ID+昵称；新人一天内不改昵称会被移出。
  • Java 版 IP：91gmc.xyz
  • 基岩版 IP：91gmc.xyz，端口：11451

群内绑定白名单文本

Java版：绑定java版+你的游戏id

基岩版：绑定基岩版+你的游戏id

例如：群内发送 绑定java版Zhongzi2022，或先进服后发送 绑定基岩版caixukun，可以绑定游戏白名单。

基岩版务必先进入大厅再绑定白名单。如果未先进服直接发送绑定文本，通常不会生效。

基岩 Java 双绑方法

  1. 先清空基岩版一切贵重物品。
  2. 登录 Java 版并输入：/linkaccount 基岩版ID
  3. 按系统提示完成绑定，双绑后请绑定 Java 版白名单。

忘记密码可联系种子重置（5 元）。

二、领地系统

先用木锄分别左右键选定区域两角，再建立领地（每格 0.5 金币）。

/res create <领地名字> 建立领地
/res ? 列出全部领地指令
/res set 设置领地权限（需站在自己领地内）
/res remove 删除领地
/res add <领地名> player <玩家名> <权限标志> 给予玩家特定领地权限
/res remove <领地名> player <玩家名> <权限标志> 移除玩家特定领地权限
/res trust <领地名> <玩家名> 将领地所有权赋予玩家
/res untrust <领地名> <玩家名> 移除领地所有权
/res list 列出您拥有的所有领地
/res info 查看您所在领地详细信息
/res settp <领地名> 设置领地传送点
/res tpa <领地名> 传送到领地传送点
/res listtp 列出所有领地传送点
/res areashow <领地名> 显示领地边界
/res areahide <领地名> 隐藏领地边界
/res expand <领地名> <长度> 扩大自己的领地

四、特殊试炼机制：符文之地

进入方式

  1. 先寻找悬空的符文之岛。
  2. 可从正面入口进入，也可向下挖掘进入折越门。
  3. 门后试炼由基岩层包裹，不从入口进入则无法正常进入试炼区。
进入符文之地后，无法通过传送门、地狱门、珍珠传送装置或传送命令返回。

危险机制

  • 床、重生锚、末影箱在此维度会爆炸。
  • 死亡时有 20% 概率保留全部背包返回主世界。
  • 死亡时有 80% 概率清空背包返回主世界，物品无法寻回。

试炼与迁跃碎片

试炼内灾厄宝库有极低概率刷出迁跃碎片，符文之地怪物强度很高，请规划装备。
  • 副手持迁跃碎片死亡时，可保留主手物品返回主世界，其余物品和盔甲清空。
  • 主手持迁跃碎片时同理，可保留副手物品。
  • 迁跃碎片完成一次迁跃后会碎裂。
  • 推荐使用潜影盒携带关键资源。
